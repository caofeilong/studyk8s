
chmod +x   修改文件执行权限

注意事项：

1. 变量定义`=`号的两边不能用空格  eg: name=caofeilong
2. 将命令的值赋值给一个变量   
eg: 
    a=`date`
    b=${ls -l}   v=$b



变量的使用方法
1.  $name  在变量名前面加 `$` 符号
2.  ${name}
3.  **双引号**可以引用变量 **单引号**直接输出


`set` 列出所有的名称
`uset name` 删除变量

特殊变量

1.  `$0`  执行的命令
2.  `$1……$N` 变量的参数
3.  `$*` 程序的所有参数
4.  `$#` 程序参数的个数
5.  `$$`  程序的pid
6.  `$!`  执行上一个指令的程序的pid
7.  `$?`  执行上一个命令的返回值
